<p>Aspose.Cells Cloud SDK per Node.js consente alle tue applicazioni basate su cloud di leggere fogli di calcolo Excel, esportare cartelle di lavoro XLSX, convertire Microsoft Excel e documenti OpenOffice in PDF, XPS, TIFF, Markdown (MD), DIF, SVG, XML (SpreadsheetML), CSV , TSV, HTML e altri formati supportati, senza installare alcun software di terze parti.</p>
<p>Aspose.Cells Cloud SDK for Node.js ottiene la sua robusta funzionalità dal sottostante Aspose.Cells REST API e consente al tuo codice Node.js di funzionare con Excel formattazione condizionale, filtri automatici, forme automatiche, grafici, OleObjects, interruzioni di pagina, ListObjects, tabelle pivot, compiti e gamme. Utilizza semplici comandi Node.js per creare, aggiornare, crittografare, unire, proteggere con password, esportare e convertire in modo programmatico file XLS, XLSX, XLSB, XLSM, XLT, XLTX, XLTM e ODS in altri formati supportati.</p>

    <h2 class="h2title">
 Inizia con Excel Spreadsheet Processing SDK
    </h2>
    <p>
 Per iniziare con Aspose.Cells Cloud SDK for Node.js, non è necessario installare nulla. Tutto quello che devi fare è creare un account su
     <a href="https://dashboard.aspose.cloud/#/apps">
 Aspose per Nuvola
     </a>
 ottenere le informazioni sull'applicazione. Una volta ottenuto l'App SID &amp; key, sei pronto per utilizzare Aspose.Cells Cloud SDK for Node.js su qualsiasi piattaforma che supporti le API REST. Aspose.Cells Cloud SDK for Node.js offre tutte le funzionalità per creare, modificare e convertire fogli di calcolo. Consente inoltre di estrarre i dati dai fogli di calcolo per un'ulteriore elaborazione o archiviazione.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
 Elaborazione cartelle di lavoro via Cloud SDK
    </h2>
    <p>
 Aspose.Cells Cloud SDK per Node.js consente di creare, dividere, unire, cercare e sostituire testo, protezione tramite password, esportazione nel formato di file supportato, lettura degli elementi di testo della cartella di lavoro, adattamento automatico delle righe della cartella di lavoro ignorando gli elementi nascosti, crittografia della password della cartella di lavoro e fornitura lunghezza della chiave di crittografia, importare i dati nella cartella di lavoro, calcolare tutte le formule nella cartella di lavoro, impostare le impostazioni di calcolo della formula fornendo una strategia di precisione ecc., utilizzare l'indicatore intelligente, convertire la cartella di lavoro dal contenuto della richiesta in un altro formato, applicare la password per consentire la modifica del contenuto.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
Cloud SDK per elaborare i fogli di lavoro
    </h2>
    <p>
Utilizzando Aspose.Cells Cloud SDK for Node.js puoi eliminare una riga specifica o tutte le righe nel foglio di lavoro, recuperare i dati della cella del foglio di lavoro o lo stile della cella in base al nome della cella, leggere i dati della riga o della colonna del foglio di lavoro in base al suo indice, calcolare la formula della cella, cancellare il contenuto della cella, copiare cella in un'altra cella, copia colonne e righe del foglio di lavoro, raggruppa righe e colonne del foglio di lavoro, nascondi righe e colonne del foglio di lavoro, imposta gli stili di colonne e righe del foglio di lavoro, imposta il valore HTMLSTRING nella cella, imposta il valore dell'intervallo di celle e modifica molte altre impostazioni del foglio di lavoro.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
 Sfrutta la potenza di Pivot Tables &amp; Intervalli
    </h2>
    <p>
 Aspose.Cells Cloud SDK per Node.js consente di aggiungere una tabella pivot nel foglio di lavoro, aggiungere o eliminare campi pivot nella tabella pivot, aggiungere o rimuovere il filtro pivot per l'indice della tabella pivot, aggiornare lo stile della cella per la tabella pivot e calcolare i dati della tabella pivot nelle celle .
    </p>
    <p>
Aspose.Cells Cloud SDK per Node.js può anche essere utilizzato per ottenere l'elenco delle celle in un intervallo in base al nome dell'intervallo o agli indici delle colonne delle righe, impostare la larghezza della colonna dell'intervallo, combinare un intervallo di celle in una singola cella, spostare l'intervallo corrente nell'intervallo di destinazione , imposta il bordo del contorno attorno a un intervallo di celle., imposta l'altezza della riga dell'intervallo, imposta lo stile dell'intervallo, separa le celle unite di questo intervallo, inserisci un valore nell'intervallo e converti il suo tipo di dati e copia l'intervallo nel foglio di lavoro.
    </p>
