<p>Aspose.Cells Cloud SDK for Android 助您开发 Microsoft Excel （XLS、XLSX、XLSB、XLSM、XLT、XLTX、XLTM）和 OpenOffice (ODS) 电子表格处理应用程序，适用于 Android 设备。 Aspose.Cells 适用于 Android 的 Cloud SDK 根据 MIT 许可提供，并基于 Aspose.Cells REST API。</p>
<p>Aspose.Cells 适用于 Android 的 Cloud SDK 使您的 Android 代码能够访问云电子表格文档并操作工作簿、工作表、单个单元格、行和列。您的 Android 程序将能够从工作表图片中提取条形码、应用自动过滤器（日期过滤器、图标过滤器、动态过滤器）、访问图表、应用条件格式以及以编程方式配置电子表格页面设置。</p>
<p>Aspose.Cells 适用于 Android 的 Cloud SDK 允许您在 Excel 和 ODS 工作表中使用 OleObjects、ListObjects、数据透视表、范围、形状和任务，而无需安装任何第三方软件。</p>
<p>Aspose.Cells 适用于 Android 的 Cloud SDK 还可以帮助您为基于 Android 的移动设备创建电子表格转换器应用程序，因为它允许将 Microsoft Excel、OpenOffice、SpreadsheetML、CSV、TSV、TXT（TabDelimited）、HTML、MHTML、SXC、FODS 转换为其他支持的格式，例如 PDF、XPS、TIFF、SVG、DIF 和 MD (Markdown)。</p>
