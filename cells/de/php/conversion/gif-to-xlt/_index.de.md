Das Konvertieren von Dateiformaten von GIF in XLT ist eine komplexe Aufgabe. Alle GIF-zu-XLT-Formatübergänge werden von unserem PHP SDK durchgeführt, während der strukturelle und logische Hauptinhalt der GIF-Quelltabelle beibehalten wird. Unsere PHP-Bibliothek ist eine professionelle Lösung, um GIF-Dateien online in XLT-Dateien umzuwandeln. Dieses Cloud-SDK bietet PHP-Entwicklern leistungsstarke Funktionen und perfekte XLT-Ausgabe.

// For complete examples and data files, please go to https://github.com/aspose-cells-cloud/aspose-cells-cloud-php/
    <?php
    require_once('vendor\autoload.php');
    use \Aspose\Cells\Cloud\Api\CellsApi;
    $instance = new CellsApi(getenv("ProductClientId"),getenv("ProductClientSecret"));
    $path ='Book1.gif';    
    $format ='xlt';
    $password = null;

    $result = $this->instance->cellsWorkbookPutConvertWorkBook($path ,$format, $password,  $outPath);
    $size = $result->getSize();

    fclose($file);
