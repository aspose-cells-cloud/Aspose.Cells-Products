Das Konvertieren von Dateiformaten von TSV in SQL ist eine komplexe Aufgabe. Alle Formatübergänge von TSV zu SQL werden von unserem NodeJS SDK durchgeführt, während der strukturelle und logische Hauptinhalt der TSV-Quelltabelle beibehalten wird. Unsere NodeJS-Bibliothek ist eine professionelle Lösung zur Online-Konvertierung von TSV- in SQL-Dateien. Dieses Cloud SDK bietet NodeJS-Entwicklern leistungsstarke Funktionen und eine perfekte SQL-Ausgabe.

// For complete examples and data files, please go to https://github.com/aspose-cells-cloud/aspose-cells-cloud-node/

    const localPath = "../TestData/source/";
    var fs = require('fs');
    var path = require('path');
    const cellsApi = new CellsApi(process.env.ProductClientId, process.env.ProductClientSecret);
    var req = new CellsWorkbook_PutConvertWorkbookRequest({
        file: fs.createReadStream(localPath + "datasource.tsv"),
        format: "sql",
    });
    cellsApi.cellsWorkbookPutConvertWorkbook(req)

        console.log(result)
    });
