Das Speichern von Dateiformaten aus SXC als PDF ist eine komplexe Aufgabe. Alle SXC-Formatübergänge in das PDF-Format werden von unserem Go SDK durchgeführt, wobei der strukturelle und logische Hauptinhalt der Quell-SXC-Tabelle erhalten bleibt. Unsere Go-Bibliothek ist eine professionelle Lösung zum Online-Speichern von SXC-Dateien im Format PDF.
