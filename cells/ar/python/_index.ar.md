<p>Aspose.Cells Cloud SDK لـ Python يساعدك على تطوير تطبيقات منشئ جداول البيانات السحابية والمشاهد والدمج والتقسيم والتشفير والمحول بلغة برمجة Python. يتم توزيع Aspose.Cells Cloud SDK لـ Python بموجب ترخيص MIT ويتم تشغيله بواسطة Aspose.Cells REST API الأساسي.</p>
<p>Aspose.Cells Cloud SDK لـ Python يسمح لكود Python الخاص بك بأداء وظائف المستوى الأساسي إلى المستوى المتقدم على Microsoft Excel وجداول بيانات OpenOffice المستضافة على التخزين السحابي. استخدم برنامج Python للعمل مع الجداول المحورية ، والعمل مع XML (SpreadsheetML) ، وقراءة Excel الملفات في السحابة via Python ، وتحويل XLSX إلى HTML ، CSV ، PDF ، XPS ، MD (Markdown) ، تنسيقات DIF ، TIFF وغيرها من التنسيقات المدعومة.</p>
