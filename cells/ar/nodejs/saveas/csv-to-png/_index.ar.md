يعد حفظ تنسيقات الملفات من CSV بالرقم PNG مهمة معقدة. يتم تنفيذ جميع التحولات من تنسيق CSV إلى PNG بواسطة NodeJS SDK الخاص بنا مع الحفاظ على المحتوى الهيكلي والمنطقي الرئيسي لجدول بيانات CSV المصدر. تعد مكتبة NodeJS الخاصة بنا حلاً احترافيًا لحفظ ملف CSV كملفات PNG عبر الإنترنت. يوفر Cloud SDK لمطوري NodeJS وظائف قوية وإخراج PNG مثاليًا.

{{% blocks/products/cells/cells-cloud-noreplacecode title="مثال على رمز NodeJS لحفظ ملف CSV كـ PNG باستخدام REST API" gistPath="" %}}
  
```js
// For complete examples and data files, please go to https://github.com/aspose-cells-cloud/aspose-cells-cloud-node/
    const { CellsApi, CellsSaveAs_PostDocumentSaveAsRequest } = require("asposecellscloud");
    const cellsApi = new CellsApi(process.env.ProductClientId, process.env.ProductClientSecret);
    var req = new CellsSaveAs_PostDocumentSaveAsRequest({
      name: "Book1.csv",
      folder: "CellsTests",
      newfilename: "Book1Saveas.png",
    });
    cellsApi.cellsSaveAsPostDocumentSaveAs(req)
      .then((result) => {
        console.log(result)
    });
```
