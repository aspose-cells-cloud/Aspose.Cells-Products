<p>Aspose.Cells Cloud SDK for Swift assists the programmers in seamlessly integrating Microsoft Excel & OpenOffice spreadsheet view and processing features into their cloud-based applications developed in Swift. Aspose.Cells Cloud SDK for Swift is licensed under MIT.</p>
<p>Aspose.Cells Cloud SDK for Swift enables your Swift apps to retrieve spreadsheets, worksheets and workbooks stored at the cloud as well as generate new spreadsheet. You can also tweak you Swift code to process and modify conditional formatting, charts, tasks, shapes, filters, OleObjects, ranges, ListObjects, and pivot tables, from within your applications without installing any 3rd party software.</p>
<p>Use Aspose.Cells Cloud SDK for Swift to convert Excel (XLS, XLSX, XLSB, XLSM, XLT, XLTX, XLTM) & OpenOffice (ODS) worksheets to other popular formats, such as, XML (SpreadsheetML), TSV, CSV, TXT (TabDelimited), HTML, MHTML, DIF, TIFF, SVG, XPS, PDF & Markdown (MD). You can also work with the SXC and FODS formats using Aspose.Cells Cloud SDK for Swift.</p>
