<p>Aspose.Cells Cloud SDK for CPP helps developers to create their own cloud based spreadsheet creator, reader, viewer, and converter applications using Go language. The SDK provides an easy-to-use interface for Aspose.Cells REST API to perform a wide range of spreadsheet processing tasks such as creation, modification, conversion, and rendering of Excel documents in the cloud.</p>
<p>Aspose.Cells Cloud SDK for CPP fully supports several advanced features related to spreadsheets reading and writing such as create a new spreadsheet from the scratch, modify an existing file, add new worksheet, import data to an Excel worksheet, hide and un-hide a worksheet, move a worksheet, sort data in a worksheet, copy or rename a sheet, freeze or unfreeze a worksheet, add or remove a workbook, encrypt or decrypt a workbook, password protect an Excel workbook, merge excel workbooks and many more.</p>
<p>Using Aspose.Cells Cloud SDK for CPP developer can easily accomplish auto filtering, pivot tables handling, and conditional formatting to spreadsheets using CPP commands. The SDK has provided a very powerful spreadsheet file conversion operation and fully support inter-conversion between Microsoft Excel formats like XLS, XLSX, XLSB, XLSM, XLT, XLTX, XLTM as well as conversion to other popular file formats like PDF, XPS, SVG, TIFF, Markdown & DIF formats.</p>
