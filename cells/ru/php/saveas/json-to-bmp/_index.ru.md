Сохранение формата файлов из JSON как BMP — сложная задача. Все переходы формата JSON в формат BMP выполняются нашим SDK PHP с сохранением основного структурного и логического содержимого исходной электронной таблицы JSON. Наша библиотека PHP — это профессиональное решение для сохранения JSON в виде файлов BMP онлайн. Этот Cloud SDK предоставляет разработчикам PHP мощные функциональные возможности и идеальный результат BMP.

{{% blocks/products/cells/cells-cloud-noreplacecode title="PHP Пример кода для сохранения JSON как BMP с использованием REST API" gistPath="" %}}
  
```php
// For complete examples and data files, please go to https://github.com/aspose-cells-cloud/aspose-cells-cloud-php/
    $apiInstance = new CellsApi( getenv("ProductClientId"),getenv("ProductClientSecret") );
    $name ='Book1.json';
    $saveOptions =new \Aspose\Cells\Cloud\Model\SaveOptions();
    $saveOptions->SaveFormat = "bmp";
    $newfilename = "Book1Saveas.bmp";
    $isAutoFitRows= 'true';
    $isAutoFitColumns= 'true';
    $folder = "Temp";
    $result = $apiInstance->cellsSaveAsPostDocumentSaveAs($name, $saveOptions, $newfilename,$isAutoFitRows, $isAutoFitColumns, $folder);
```
