<p>Aspose.Cells Cloud SDK для Node.js позволяет вашим облачным приложениям читать электронные таблицы Excel, экспортировать книги XLSX, преобразовывать Microsoft Excel и документы OpenOffice в PDF, XPS, TIFF, Markdown (MD), DIF, SVG, XML ( Электронная таблицаML), CSV , TSV, HTML и другие поддерживаемые форматы без установки какого-либо стороннего программного обеспечения.</p>
<p>Aspose.Cells Облачный SDK для Node.js получает надежную функциональность от базового Aspose.Cells REST API и позволяет вашему коду Node.js работать с Excel условным форматированием, автоматическими фильтрами, автоматическими формами, диаграммами, OleObjects, разрывами страниц, ListObjects, сводными таблицами, задачи и диапазоны. Используйте простые команды Node.js для программного создания, обновления, шифрования, объединения, защиты паролем, экспорта и преобразования файлов XLS, XLSX, XLSB, XLSM, XLT, XLTX, XLTM и ODS в другие поддерживаемые форматы.</p>

    <h2 class="h2title">
 Начните работу с Excel SDK для обработки электронных таблиц
    </h2>
    <p>
Чтобы начать работу с Aspose.Cells Cloud SDK для Node.js, вам не нужно ничего устанавливать. Все, что вам нужно сделать, это создать учетную запись на
     <a href="https://dashboard.aspose.cloud/#/apps">
 Aspose для облака
     </a>
 и получите информацию о приложении. Получив SID приложения &amp; key, вы готовы использовать Aspose.Cells Cloud SDK для Node.js на любой платформе, поддерживающей REST API. Aspose.Cells Cloud SDK для Node.js предоставляет все возможности для создания, редактирования и преобразования электронных таблиц. Он также позволяет извлекать данные из электронных таблиц для дальнейшей обработки или хранения.
    </p>

   <div class="col-lg-12">
    <h2 class="h2title">
 Обработка книг via Cloud SDK
    </h2>
    <p>
Aspose.Cells Cloud SDK для Node.js позволяет создавать, разделять, объединять, искать и заменять текст, защищать паролем, экспортировать в поддерживаемый формат файла, читать текстовые элементы книги, автоматически подбирать строки книги, игнорируя скрытые элементы, шифровать пароль книги и предоставлять длина ключа шифрования, импорт данных в рабочую книгу, вычисление всех формул в рабочей книге, установка параметров расчета формулы путем предоставления стратегии точности и т. д., использование интеллектуального маркера, преобразование рабочей книги из содержимого запроса в другой формат, применение пароля для разрешения модификации содержимого.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
Cloud SDK для обработки листов
    </h2>
    <p>

    </p>
   </div>
   <div class="col-lg-12">

 Используйте возможности сводных таблиц &amp;amp;amp;amp;amp;amp; Диапазоны

 Aspose.Cells Cloud SDK для Node.js позволяет добавлять сводную таблицу на лист, добавлять или удалять поля сводной таблицы в сводной таблице, добавлять или удалять фильтр сводной таблицы для индекса сводной таблицы, обновлять стиль ячеек для сводной таблицы и вычислять данные сводной таблицы по ячейкам. .
    </p>
    <p>
Aspose.Cells Cloud SDK для Node.js также можно использовать для получения списка ячеек в диапазоне по имени диапазона или индексам столбца строки, установке ширины столбца диапазона, объединении диапазона ячеек в одну ячейку, перемещении текущего диапазона в целевой диапазон , устанавливает границу контура вокруг диапазона ячеек., устанавливает высоту строки диапазона, устанавливает стиль диапазона, разъединяет объединенные ячейки этого диапазона, помещает значение в диапазон и преобразует его тип данных, а также копирует диапазон на рабочий лист.
