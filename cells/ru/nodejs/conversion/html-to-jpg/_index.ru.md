Преобразование форматов файлов HTML в JPG может оказаться сложной задачей. Наш NodeJS SDK обрабатывает все преобразования формата HTML в JPG, сохраняя при этом основное структурное и логическое содержимое исходной таблицы HTML. Наша библиотека NodeJS предоставляет профессиональное решение для онлайн-конвертации HTML в файлы JPG. Этот Cloud SDK предоставляет разработчикам NodeJS мощные функциональные возможности и обеспечивает высококачественный вывод в формате JPG.

{{% blocks/products/cells/cells-cloud-noreplacecode title="Пример кода NodeJS для преобразования HTML в JPG с использованием Cells Cloud SDK" gistPath="" %}}
 
```js
// For complete examples and data files, please go to https://github.com/aspose-cells-cloud/aspose-cells-cloud-node/
    const { CellsApi, CellsWorkbook_PutConvertWorkbookRequest } = require("asposecellscloud");
    const localPath = "../TestData/source/";
    var fs = require('fs');
    var path = require('path');
    const cellsApi = new CellsApi(process.env.ProductClientId, process.env.ProductClientSecret);
    var req = new CellsWorkbook_PutConvertWorkbookRequest({
        file: fs.createReadStream(localPath + "datasource.html"),
        format: "jpg",
    });
    cellsApi.cellsWorkbookPutConvertWorkbook(req)
        .then((result) => {
        console.log(result)
    });
```
