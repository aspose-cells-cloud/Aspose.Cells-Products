Сохранение форматов файлов из CSV в формате XLSM — сложная задача. Все переходы формата CSV в XLSM выполняются нашим Ruby SDK с сохранением основного структурного и логического содержимого исходной электронной таблицы CSV. Наша библиотека Ruby — это профессиональное решение для сохранения файлов CSV в формате XLSM онлайн. Этот Cloud SDK предоставляет разработчикам Ruby мощные функциональные возможности и идеальный вывод XLSM.

    newfilename = 'Book1Saveas.xlsm'
    is_auto_fit_rows = true
