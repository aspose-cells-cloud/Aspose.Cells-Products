Сохранение форматов файлов с PNG как PDF — сложная задача. Все переходы форматов с PNG на PDF выполняются нашим Go SDK с сохранением основного структурного и логического содержимого исходной таблицы PNG. Наша библиотека Go — это профессиональное решение для сохранения файлов PNG как PDF в Интернете. Этот Cloud SDK предоставляет разработчикам Go мощные функциональные возможности и идеальный вывод PDF.

{{% blocks/products/cells/cells-cloud-noreplacecode title="Пример кода Go для сохранения PNG как PDF с использованием REST API" gistPath="" %}}
  
```go
// For complete examples and data files, please go to https://github.com/aspose-cells-cloud/aspose-cells-cloud-go/
    instance := asposecellscloud.NewCellsApiService(os.Getenv("ProductClientId"), os.Getenv("ProductClientSecret"))
    saveAsOpts := new(asposecellscloud.CellsSaveAsPostDocumentSaveAsOpts)
    saveAsOpts.Name = "Book1.png"
    saveAsOpts.Folder = "CellsTests"
    saveAsOpts.Newfilename = "Book1SaveAs.pdf"
    _, _, err1 := instance.CellsSaveAsPostDocumentSaveAs(saveAsOpts)
    if err1 != nil {
	    println(err1)
    }
```
