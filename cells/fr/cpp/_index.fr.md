<p>Aspose.Cells Cloud SDK for CPP aide les développeurs à créer leurs propres applications cloud de création, de lecture, de visualisation et de conversion de feuilles de calcul à l'aide du langage Go. Le SDK fournit une interface facile à utiliser pour Aspose.Cells REST API pour effectuer un large éventail de tâches de traitement de tableur telles que la création, la modification, la conversion et le rendu de Excel documents dans le cloud.</p>
<p>Aspose.Cells Le SDK Cloud pour CPP prend entièrement en charge plusieurs fonctionnalités avancées liées à la lecture et à l'écriture de feuilles de calcul, telles que la création d'une nouvelle feuille de calcul à partir de zéro, la modification d'un fichier existant, l'ajout d'une nouvelle feuille de calcul, l'importation de données dans une feuille de calcul Excel, le masquage et l'affichage d'une feuille de calcul, déplacer une feuille de calcul, trier des données dans une feuille de calcul, copier ou renommer une feuille, geler ou dégeler une feuille de calcul, ajouter ou supprimer un classeur, chiffrer ou déchiffrer un classeur, protéger par mot de passe un classeur Excel, fusionner des classeurs Excel et bien d'autres.</p>
<p>L'utilisation de Aspose.Cells Cloud SDK pour le développeur CPP peut facilement effectuer le filtrage automatique, la gestion des tableaux croisés dynamiques et la mise en forme conditionnelle des feuilles de calcul à l'aide des commandes CPP. Le SDK a fourni une opération de conversion de fichier de feuille de calcul très puissante et prend entièrement en charge l'inter-conversion entre les formats Microsoft Excel comme XLS, XLSX, XLSB, XLSM, XLT, XLTX, XLTM ainsi que la conversion vers d'autres formats de fichiers populaires comme PDF, XPS, SVG , TIFF, formats Markdown et DIF.</p>
